ThinkDesk Tenancy REST API

O ThinkDesk é uma API REST multi-inquilino (multi-tenant) projetada para gerenciar um sistema de suporte e atendimento (ITSM). A API permite que diferentes "inquilinos" (tenants) operem de forma isolada, cada um com seus próprios usuários, técnicos, tickets e políticas de serviço.

Arquitetura Limpa

Domain

O coração da aplicação. Contém as entidades de negócio, regras e interfaces dos repositórios. Totalmente independente de frameworks.

Application

Orquestra o fluxo de dados. Contém os Controllers que expõem a API REST e os DTOs que definem os contratos de dados (JSONs).

Infra

A camada externa. Contém as implementações concretas (JPA), configuração de segurança (Spring Security), e tratamento de exceções.

Guia de Endpoints da API
